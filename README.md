# Waterpump

Management of Pumps with Django and React/ReactNative

0. python / node install
1. pip install pipenv
1. pipenv shell
1. pipenv install -r requirements/local.txt
1. pipenv install django
1. pgadmin install
1. name: waterpump, localhost, port 5432, 9315, waterpump_db
1. frontend: .env NODE_PATH=src
1. python manage.py makemigrations, migrate, createsuperuser
1. python manage.py runserver
1. frontend: npm i, yarn build, yarn start
